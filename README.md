- [rapidfuzz](https://github.com/maxbachmann/RapidFuzz): fuzzy string matching; alternative to `fuzzywuzzy`

## Logging

- [loguru](https://github.com/Delgan/loguru): sensible logging defaults; alternative to `logging`
